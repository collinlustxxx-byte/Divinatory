# Application Divinatory

L'application Divinatory est un outil qui permet d'explorer le monde de la divination. Elle offre diverses méthodes de divination, telles que la cartomancie, l'astrologie et d'autres pratiques mystiques pour aider les utilisateurs à découvrir des réponses et des orientations dans leur vie.

## Fonctionnalités
- Cartes de tarot interactives
- Calculs astrologiques basés sur la date de naissance
- Interprétations de symboles et de rêves

## Installation
Pour installer l'application Divinatory, clonez le dépôt et installez les dépendances avec `npm install`.

## Utilisation
Lancez l'application avec la commande `npm start` et suivez les instructions à l'écran.
